# True Abstract 5D Engine

A true abstract 5D engine is a configurable graph runtime.

Principles:
- Everything is a world.
- Every world is a graph.
- Every graph has dimensions.
- Every node has state.
- Every edge has meaning.
- Every behavior is a policy.
- Every runtime integration is an adapter.
- Every action is proposed before mutation.

Default 7D mapping:
- x = identity
- y = dependency
- z = runtime_depth
- u = world_membership
- v = machine_membership
- w = network_replication
- t = time_version_history
